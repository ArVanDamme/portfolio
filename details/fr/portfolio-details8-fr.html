Projet de groupe réalisé en 2 mois. Le but était de créer une application mobile permettant de noter des restaurants. Y ajouter des photos et pouvoir via un système de géolocalisation trouver les restaurants les plus proches de soi. Utilisation d'une API pour récupérer les données des restaurants.
